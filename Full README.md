# 📌 String Calculator - TDD Approach

This project implements a **String Calculator** using **Test-Driven Development (TDD)** in JavaScript.

## 🚀 How to Start and Run This Project

1️⃣ Clone the Repository
git clone <your-github-repo-url>
cd string-calculator

2️⃣ Install Dependencies
Run the following command to install the necessary packages:
npm install

3️⃣ Run the Tests
Since this project follows TDD, it includes automated tests.
To run all tests, execute:
npm test
